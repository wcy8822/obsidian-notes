### 📊 三种验证方式

#### 1. **快速验证** - 实时监控面板（推荐每天使用）

```bash
.claude/monitor.sh
```

**显示内容：**

- ✅ 合规率（目标：100%）
- ✅ 最近操作及备份状态
- ✅ 操作类型统计
- ✅ 彩色可视化

#### 2. **详细分析** - 合规性验证工具（推荐每周使用）

```bash
python3 .claude/verify_backup_compliance.py
```

**功能：**

- ✅ 逐条验证备份完整性
- ✅ 列出缺失/无效备份
- ✅ 导出 JSON 报告
- ✅ 按操作类型统计

#### 3. **状态查询** - 回滚系统状态

```bash
.claude/rollback.sh status
.claude/rollback.sh list
```

---

### 🔐 验证标准

**✅ 完全合规（目标）：**

- 合规率 = 100%
- 所有操作都有备份 ID
- 所有备份文件存在且完整

**⚠️ 部分合规：**

- 合规率 >= 80%
- 需要调查缺失的备份

**❌ 不合规：**

- 合规率 < 80%
- 需要立即修复备份系统

### 💡 建议的使用流程

1. **每天开始工作前**：
    
    ```bash
    .claude/monitor.sh  # 快速查看昨天的合规率
    ```
    
2. **Claude 工作时（可选）**：
    
    ```bash
    .claude/monitor.sh --watch  # 在另一个终端实时监控
    ```
    
3. **每次 Claude 会话后**：
    
    ```bash
    python3 .claude/verify_backup_compliance.py  # 详细验证
    ```
    
4. **每周生成报告**：
    
    ```bash
    python3 .claude/verify_backup_compliance.py --export
    ```
