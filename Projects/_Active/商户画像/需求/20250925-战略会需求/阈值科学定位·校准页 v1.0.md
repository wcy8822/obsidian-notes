# 1. 目的

用“**稳定性 / 经济性 / 一致性**”三把尺子，给出**样本阈值 N\_min、差异阈值 δ\_bp、回收率档位**的**科学定位**，并固化为版本化口径；同时提供**校准页模板**，便于 DS 或我们自行复用。

> 结论说法：**“我们选的是让结论‘最小代价变稳’的拐点阈值。”**

---

# 2. 默认阈值 v1.0（先跑一版）

* **样本阈值 N\_min（主结论）**：30；**展示阈值**：10。
* **差异阈值 δ\_bp**：重叠 30bp；品牌 50bp。
* **回收率场景**：30% / **50%（默认）** / 70%。
* **数据质量闸门**：覆盖率 < 80% 或 未知=99 > 20% → 降级观察。
* **最小票面 U\_min**：5 万（或 GMV×0.5% 二选一）。

> 以上为 v1.0 起跑线；下节校准后可微调并写入版本说明。

# 3. 校准方法（10 分钟跑完一轮）

## 3.1 网格（Grid）

* `N_min ∈ {20, 30, 40, 50}`
* `δ_overlap ∈ {20, 30, 40}bp`；`δ_brand ∈ {40, 50, 60}bp`
* 回收率固定三档：30/50/70%

## 3.2 三项评估指标（三把尺子）

1. **统计稳定性**（Stability）

   * 方法：对每个“同城×同品牌”对照（重叠）和“同城相邻档”对照（品牌），
     用 **bootstrap=200 次**抽样，计算 `ΔP50` 的 **95% 置信区间**宽度 `W`。
   * 指标：`稳定率 = P(W ≤ 20bp)`（或按你设定的阈值）。
2. **经济显著性**（Economics）

   * 指标：`经济率 = P(uplift_conservative ≥ U_min)`，`U_min` 默认为 5 万（或 GMV×0.5%）。
3. **一致性**（Consistency）

   * 指标：`Jaccard = |TopK(当前) ∩ TopK(邻近格点)| / |并集|`（建议 `K=20`）。

> 说明：`uplift_conservative = min(ΔP50×GMV, fee_gap_P50×GMV×回收率50%)`。

## 3.3 选拐点（Pick）

* 选择**同时满足**：`稳定率 ≥ 70%` & `经济率 ≥ 60%` & `Jaccard ≥ 0.8` 的**最小**参数组合（更省样本、更易解释）。
* 记录为 **阈值 v1.x**；其余格点入“灵敏度附录”。

# 6. 风险与止损

* 样本稀疏：`N<10` 不入任何结论；`10≤N<30` 仅展示。
* 数据质量：覆盖率<80% 或 未知=99>20% 的省/城 → 观察区。
* 解释纪律：**横截面快照≠因果**；红绿灯仅作“优先级”决策，不作为因果陈述。

# 7. 定义注释（新增）

* **bp**：基点，1bp=0.01%。
* **ΔP50**：两组中位数之差（典型差异）。
* **bootstrap**：有放回抽样重复计算统计量，形成分布以估计置信区间。
* **Jaccard 相似度**：两个集合交集/并集，衡量名单稳定性。
* **U\_min（最小票面）**：低于该收益门槛的不进入执行清单，避免“忙而不值”。
* **uplift\_conservative**：`min(ΔP50×GMV, fee_gap_P50×GMV×回收率50%)`，用于主排序。
