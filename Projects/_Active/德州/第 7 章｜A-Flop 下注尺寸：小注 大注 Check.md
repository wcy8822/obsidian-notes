## 1. 一句话结论

**Flop 用小注 / 大注 / check，本质是在用下注尺寸表达「范围优势 + 坚果优势 + 极化程度」：只要尺寸和这三个维度对上号，你就是在靠近 GTO，同时稳定 exploit 玩家池。**

## 2. 章节目的（为何存在）

这一章解决三个具体问题：

1. **为什么有些牌面要全范围小注，有些要大注极化，有些反而要大量 check？**
    
2. **在不同底池类型（SRP / 3BP / 4BP）下，小注 / 大注 / check 的“默认答案”分别是什么？**
    
3. **你在实战中如何用一套固定 checklist，在 5 秒之内选出合适的尺寸，而不是凭感觉？**
    

信息来源主要是：

- 《全范围 C-Bet 策略指南 I》关于 SB 3bet vs BTN 在 AsQc4h 全范围小注、以及 BB vs UTG JsTc5s→3♠ 全范围 30% 的例子

## 3. GTO 基线（结构来自 GTO+GO）

### 3.1 全范围小注：范围优势 + nuts 对称

GTO 环境下，只要满足：

- 进攻方整体 equity 明显领先（一般 55–65% 区间）；
    
- 顶端 nuts 组合双方都有，或差距不大；
    
- 牌面相对干燥 / 中度连接，未来牌面不剧烈翻转 nuts；
    

Solver 很乐意给出「全范围小注」的方案。

典型例子：

1. **3BP SB vs BTN，AsQc4h，SB Cbet**
    
    - SB 在这个牌面上同时有范围优势和坚果优势；
        
    - 报告中直接展示了 SB 在 AsQc4h 上用 1/3 底池做全范围 cbet，且像 QJo、Q9s 这种 equity 接近 50% 的牌，在下注 EV 上明显优于 check EV
        
    - 这里的核心：SB 的 equity 曲线是“从顶到底连续下降”的，几乎整个范围都适合下注。
        
2. **SRP BTN vs BB，A♠A♦4♣（GTO+GO）**
    
    - BTN 总体 equity ≈ 59% vs 41%，EV ≈ 3.5 vs 1.5，IP Cbet size = 1/3；
        
    - Solver 结果：IP Cbet 1/3 | Check = 100 / 0，简化为「全范围 Cbet」
        

在这类牌面上，GTO 的基线很简单：**全范围 1/3 小注高频，几乎不需要混 check。**

---

### 3.2 极化大注 / 超池：范围优势 + 明显 nuts 优势

当满足：

- 进攻方整体 equity 仍然略微领先或持平；
    
- 顶端 nuts 组合显著多于对手（典型是 2 对+ / set / nut straight 密集地集中在进攻方）；
    
- 牌面相对湿润，未来牌面不会严重削弱你现有的 nuts 优势；
    

Solver 会倾向：

- 用「极化范围 + 大注（2/3–150% pot）」来推进游戏。
    

典型例子：

1. **SRP BTN vs BB，AcKdQh（超池下注指南）**
    
    - BTN 整体 equity ≈ 58%，比 BB 高不少；
        
    - 顶端两对以上的组合在 BTN 范围中大约是 BB 的两倍；
        
    - Solver 给出：BTN 在 flop 上约 36% 频率使用 150% 底池超池下注，用的是极化范围
        
2. **SRP BTN vs BB，K♠Q♦T♠（GTO+GO 特征牌面）**
    
    - Equity ≈ 60% vs 40%，IP Cbet size = 2/3；
        
    - 报告中直接标注：“IP Cbet 2/3 | Check=100/0，简化 Cbet Range：全范围”，即全范围大注
        

结论：**当你既有范围优势，又有明显的 nuts 密度优势时，GTO 倾向用极化范围打大注 / 超池。**

---

### 3.3 Check-heavy：范围优势有限 / nuts 在对手那边

如果：

- 总体 equity 差距不大甚至略落后；
    
- 对手 defend 范围在这个牌面上 nuts 密度更高；
    
- 自己的顶端主要是高 showdown value 的单对，而不是极强牌；
    

Solver 会给出：

- 大量 check，甚至接近全范围 check；
    
- 只把一截极化范围（set / 两对 / 强 draw）拿出来下注（一般中大尺寸）。
    

典型例子：

1. **SRP BTN vs BB，7♠5♦4♠（GTO+GO）**
    
    - 双方 equity ≈ 50% vs 50%，但 BB 在顺子 / 两对 / set 的密度上更高；
        
    - 报告里 IP Cbet 2/3 ≈ 46%，Check ≈ 54%；
        
    - 简化 Cbet Range：Set、Overpair+、强听牌；简化 check range 里有大量中弱对子和高牌
        

---

### 3.4 特殊：4bet 底池的「全范围 1/4 小注」

在千算标准行动路线里，4BP 的 flop 策略被大幅简化为：

- **4BP IP/OP A 类牌面：Flop 全范围 cbet 1/4 pot。**
    
- vs raise：用 TP+（有 A 的牌面需要 AK）、FD、SD 直接 all-in
    

背后的 GTO 逻辑也一致：

- 4BP 的 SPR 很低（通常 < 3），整体范围已经高度收紧且接近线性；
    
- Flop 上整体 equity 差距有限，但谁先下注谁就能“抢”一截 EV；
    
- 小注可以在不暴露范围结构的前提下，推进底池和 SPR，给 turn、river 的极化 all-in 留空间。

## 4. 玩家池偏差（基于 PDF + 实战经验）

结合这些文档和常见微级别 / 线下局情况，玩家池在“尺寸 vs 牌面结构”上有几类稳定偏差：

1. **过度统一小注**
    
    - 不管是 A♠A♦4♣ 这种高干面，还是 7♠5♦4♠、J♥T♣8♥ 这种高协面 / nuts 偏向防守方的牌面，一律用 1/3 小注；
        
    - 这在《超池下注指南》中已经被暗示为“常见错误”：在 AKQ 上不敢用 150% 超池、只敢用小注，会严重压低 EV
        
2. **河牌才想极化，Flop / Turn 一律 merged 小注**
    
    - 《无法区分价值和诈唬…》强调：极化带来的 EV 优势是从 flop/turn 开始累积的，不是只在 river 体现
        
    - 玩家池习惯 flop 用 merged 小注，turn 继续中小注，到了河牌才突然“想起来极化”，导致整体线条不协调。
        
3. **Check = 放弃，check-range 完全被 capped**
    
    - 在 GTO+GO 的很多牌面上（如 Q♣7♠7♦、J♥T♣8♥）IP 的 check-range 里保留了不少 strong made hand，用来保护范围
        
    - 但实战里，很多人只把空气牌 check 掉，结果一 check 就被对手锁定为弱线。
        
4. **OOP XR 频率极低，让 IP 小注没有成本**
    
    - 不论 SRP 还是 3BP，报告里都给 OOP 设计了比较积极的 XR 线（强成牌 + 强 draw）
        
    - 玩家池 OOP 几乎不 XR，IP 的 30% 小注变成纯 free-roll：要么便宜看 turn，要么立刻赢下底池。
        
5. **4BP 不敢全范围小注 / check**
    
    - 千算给了极度简化的 4BP 标准线：IP/OP A 牌面 1/4 全范围 cbet；某些无 A 牌面全范围 check 等
        
    - 但玩家池在 4BP 翻牌时要么直接大注“心慌式保护”，要么干脆 check 弃完，让强范围白白浪费。
        

这些偏差，几乎点亮了 exploit 的全部空间。

## 5. Exploit 调整（本章可执行策略）

我们用三条硬规则，把 GTO 基线变成实战 exploit 策略。

### 5.1 Rule 1：玩家池“不会 XR” → IP 可以放心高频小注

适用条件：

- 你作为 IP，SRP/3BP 对阵盲注位；
    
- 对手 XR 频率极低（你打了几千手都几乎没见过 XR bluff）；
    

策略调整：

- 在 Axx 高干面、AAx、公对高牌面上：**放心执行全范围 1/3 小注**，甚至在一些 GTO 建议混 check 的面上，都可以轻度 over-cbet；
    
- 反而在 7♠5♦4♠、J♥T♣8♥ 这类 nuts 偏向 BB 的面上，**收缩范围，减少 bluff，保留 check-range。**
    

理由：对手 XR 太少，你小注拿不到 GTO 水平的“被惩罚概率”，等于是白送 EV。

### 5.2 Rule 2：玩家池“不会 fold 顶对” → 极化大注只 value 不 bluff

适用条件：

- 对手是典型跟注站：
    
    - flop / turn 跟注过多，river 几乎不弃任何 top pair；
        
- 你在 AKQ / KQTs / KJ5ss 这类 nuts 优势面上，按 GTO 本该有大量极化 bluff。
    

策略调整：

- **保留价值端的大注 / 超池**（两对+ / nuts draw）；
    
- **砍掉绝大部分 bluff 端**，把一部分听牌直接转为小注 / check；
    
- 让自己在大注 / 超池线上的 bluff ratio 显著低于 GTO。
    

你赚的是：对手跟太多，你减少 bluff，所有大注都是纯 value，长期印钞。

### 5.3 Rule 3：玩家池“对大注过度尊重” → 借极化大注偷底池

适用条件：

- 典型“怕钱玩家”：
    
    - 面对 2/3+ / 超池下注弃得特别多；
        
- 牌面属于你有明显 nuts 优势的一类（AKQ、KQTs、部分天顺听花面），GTO 确实允许大注 bluff 频率存在
    

策略调整：

- 在允许极化大注的牌面上：
    
    - 增加 bluff 端组合：
        
        - 无 showdown value 的 BDFD+Gutter、带好 blocker 的 air；
            
    - 降低小注频率，让对手更多面对“大注 or check”的二选一。
        

核心：**先确保你选的是 GTO 允许极化的面，再 exploit 地多 bluff。**

## 6. 行为模板（Checklist）

每次你作为翻前进攻方到达 flop，把下面这张“5 秒决策卡”过一遍：

1. **底池类型 + 位置**
    
    - SRP IP vs BB？SRP EP vs BB？3BP IP vs SB/BB？4BP？
        
    - 3BP/4BP 更偏向小注 / 全范围小注；SRP EP vs BB 要更谨慎。
        
2. **谁有范围优势？谁有 nuts 优势？**
    
    - 对照 GTO+GO 15 个特征牌面：
        
        - AAx / AKx：多半你范围 + nuts 双优势；
            
        - 中低连牌（754、T98）+ BB defend：nuts 多在 BB；
            
        - 高协面（KQTs、JTxs）：通常 IP nuts 明显多。
            
3. **按下面三类之一归类牌面：**
    
    - 类 A：范围优势 + nuts 大致对称 → 倾向全范围 1/3 小注；
        
    - 类 B：范围优势 + 明显 nuts 优势 → 倾向极化 2/3–150% 大注 / 超池；
        
    - 类 C：范围优势有限 / nuts 偏向对手 → check-heavy，仅用极化范围下注。
        
4. **这手牌在你范围里的层级？**
    
    - nuts / near nuts → 必须在下注树内（不论小注或大注线）；
        
    - 中等 value（top pair / overpair）→ 类 A 进小注线，类 B 部分混 check，类 C 多 check；
        
    - 高权益 draw → 类 A 小注，类 B 可以放大注 bluff 线，类 C 看 nuts 优势决定。
        
5. **检查自己的 check-range：**
    
    - 问自己：
        
        - “我 check 这张牌面时，范围里还有没有强牌 / 强 draw？”
            
    - 如果答案是“几乎没有”，说明你 check 线被严重 capped，要调整。

## 7. 数学推导（尺寸背后到底在算什么）

这里给两个你实战能直接用到的数学框架。

### 7.1 小注的「权益剥夺」模型

假设：

- 底池 100；
    
- 你用全范围 30% 小注（下注 30）；
    
- 对手对你这个小注的弃牌率是 f，跟注率 1−f，不加注。
    

你整条范围的 EV 近似：

> EV(小注) ≈ f × 100 + (1−f) × EV(被跟注后的局面)

对比 check：

> EV(check) ≈ EV(双方都看 turn 的局面)

当你有范围优势时（equity > 50%），哪怕 f 不高，单靠“让对手放弃那部分 equity”就已经有净收益。《全范围 C-Bet 指南》里多次强调：在 AsQc4h、JsTc5s→3♠ 这类场景，全范围 30% 小注带来的 EV 提升，主要来源不是 value vs bluff，而是**拒绝对手的 equity**

### 7.2 大注 / 超池的「bluff:value 比例」模型

当你下注 S（相对于底池 P）时，给对手的 pot odds 是：

> 对手需要的防守胜率 = S / (P + S)

例如：

- 底池 100，你下注 100（pot）：
    
    - 对手跟注需要的胜率 = 100 / (100 + 100) = 50%；
        
    - 在平衡状态下，你 bluff:value 的比例如下：
        
        - bluff / (bluff+value) ≈ 50%
            
- 底池 100，你下注 150（1.5x pot）：
    
    - 对手需要胜率 = 150 / (100+150) = 60%；
        
    - 在完全极化、对手无 exploit 能力时，你可以让 bluff 约占 60%。
        

《超池下注指南》中 AKQ 牌面 BTN 用 150% 底池下注、而双方整体 equity 接近 50%，但 BTN 在 EV 上碾压 BB，本质就是：BTN 用极化范围 + 合理 bluff:value 比例，让对手在无差别抓诈 / 过度弃牌中二选一，长期 EV 失血

实战中你不需要精确记比例，只要记住两个要点：

1. 尺寸越大，你**理论上可以越多 bluff**——前提是 nuts 真在你这边；
    
2. 玩家池要么 bluff 远少于 GTO，要么面对大注弃牌远多于 GTO——这两端都是 exploit 的金矿。

## 8. 高频场景（2–3 个落地模板）

### 场景 1：SRP BTN vs BB，高干面 AAx / AKx

- 典型牌面：A♠A♦4♣、A♣K♦5♠；GTO+GO 报告中 IP 采用全范围 1/3 小注
    
- 行动模板（IP）：
    
    - 绝大部分范围：直接 1/3 cbet；
        
    - 只保留极少数 QQ–JJ、部分 A-high 入 check-range 做保护（如果你有精力做混频）；
        
- Exploit：玩家池几乎不会 XR，你可以放心全范围小注 → 高频打印小 pots。
    

### 场景 2：SRP BTN vs BB，高协面 KQTs / AKQ

- 典型牌面：K♠Q♦T♠、AcKdQh
    
- 行动模板：
    
    - IP：构建极化范围（两对+ + 强 draw）用 2/3–150% 大注；
        
    - 一部分中等牌（KQ、top pair no kicker）混入 check；
        
- Exploit：
    
    - 对跟注站 → 大注线 value-only，几乎不 bluff；
        
    - 对怕钱玩家 → 严选带 blocker 的无 showdown bluff，用大注压。
        

### 场景 3：SRP / 3BP vs BB，中低连牌 7♠5♦4♠、J♥T♣8♥

- 这类牌面在 GTO+GO 里 OOP nuts 密度高，IP 只能用极化范围大注，check-range 保留大量强牌
    
- 行动模板：
    
    - IP：
        
        - 用 set / 两对 / nut draw 做 2/3 大注；
            
        - 大量 top pair / 中对直接 check-back。
            
    - OOP：
        
        - 明确 XR 线（强牌+强 draw），禁止“只跟不加”。

## 9. 复盘模板（本章相关）

以后你复盘任何一手 flop spot，可以按这几个问题写：

1. 这手牌当时的底池类型 / 位置 / SPR 是什么？
    
2. 这块牌面属于 A / B / C 哪一类（范围优势 + nuts 优势如何）？
    
3. GTO 的标准 size 倾向是什么（全范围小注 / 大注极化 / check-heavy）？
    
4. 我当时选了哪条尺寸线？是「线性小注」还是「极化大注」，还是 check？
    
5. 我的 check-range 里有没有足够强度？如果没有，对手如何 exploit 我？
    
6. 玩家池在这个 spot 上的典型偏差是什么（过度弃牌 / 跟注站 / 不 XR）？
    
7. 在这些偏差下，我最应该怎么 exploit？
    
8. 下一次遇到同类 spot，我准备强制执行哪条「默认线」？
    

把这 8 个问题写成固定模板，长久下来你的 flop 尺寸感会从“凭感觉”变成“有章可循”。

## 10. 常见错误（至少 5 条）

1. **统一 1/3，小注打天下**
    
    - 不分牌面结构、不分 nuts 分布，一律 BTN vs BB 30% 小注；
        
    - 结果是在 nuts 劣势面上自找麻烦，在 nuts 优势面上白白浪费 EV。
        
2. **check 就是认输**
    
    - check-range 里没有任何 strong hand，每一次 check 都等于给对手“无阻力印钞机会”。
        
3. **不会极化，永远 merged**
    
    - 在该极化大注的牌面上用一堆“top pair+听牌+空气”下注 1/2 pot，导致整条线没法在 turn / river 建立压力。
        
4. **OOP 从不 XR，IP 小注无成本**
    
    - GTO+GO 中不少 flop 都给 OOP XR 线，玩家池实战却几乎只 call，不加注；
        
    - IP 随便 1/3 小注，拿到的是“对手过度防守+不给惩罚”的超额 EV。
        
5. **4BP 瞎玩，不懂全范围 1/4 的意义**
    
    - 本该 4BP A 牌面全范围 1/4 小注，实战却直接 2/3 大注，导致范围完全失衡；
        
    - 或者完全不 cbet，让自己强范围失去主动权。
        
6. **尺寸和后续计划脱节**
    
    - flop 用 1/3 小注，turn 明明是 blank 也突然换成小 size 或直接 check，完全没有「小注 → turn 极化 → river 收割」的策略树意识。
