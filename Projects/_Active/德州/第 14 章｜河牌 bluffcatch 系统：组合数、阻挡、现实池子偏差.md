## 14.1 一句话结论

**真正的河牌 bluffcatch，不是“我感觉他在诈唬”，而是：在理解对手价值/诈唬范围与玩家池倾向的基础上，用“阻挡价值、不阻挡诈唬、还能打赢价值底端”的手牌，在数学允许的赔率下做出高波动但长期 +EV 的跟注。**

---

## 14.2 本章目的：抓诈到底要搞清楚什么？

本章只帮你解决三件事：

1. **用什么牌去抓？（牌型层面）**
    
    - 用《河牌抓诈进阶指南》的“抓诈三原则”：
        
        - 阻挡对手价值范围；
            
        - 不阻挡对手诈唬范围；
            
        - 能打赢对手价值范围中一部分最弱牌。
            
2. **在什么局面、什么玩家上抓？（结构与玩家池层面）**
    
    - 有些结构 + 玩家组合是“必须有抓诈频率”的；
        
    - 有些结构 + 玩家组合几乎应该 **0 频率抓诈**，你 call 越多输越多。
        
3. **如何用组合数 + 赔率把“感觉”变成算账？（数学层面）**
    
    - 对手下注大小 → 你需要多少胜率才能 call？
        
    - 对手可能的价值组合 vs 诈唬组合 roughly 有多少？
        
    - 在当前玩家池倾向（普遍诈唬不足 / 个别过度诈唬）下，
        
        - 这手 call 是“长期高波动 +EV”，
            
        - 还是“短期爽一次、长期输死”的坏英雄？

## 14.3 理论基线：抓诈 EV 与“非稳定胜率”

### 14.3.1 Janda：call 的 EV 怎么算？

设：

- pot = P（对手下注前的底池）；
    
- 对手在河牌下注 B；
    
- 如果你 call：
    
    - 当对手 bluff 时，你赢 P + B；
        
    - 当对手 value 时，你输 B。
        

用对手 bluff 频率 `x` 表示你 call 的 EV：

[  
EV_{call} = x \cdot (P + B) - (1-x) \cdot B  
]

当 `EV_call = 0` 时是盈亏平衡，解得你所需的最小 bluff 频率：

[  
x = \frac{B}{P + 2B}  
]

也就是：**对手 bluff 的比例，只要高于这个阈值，你的 call 就是 +EV 的。**

举例：

- pot = 100，villain bet 100（pot size）：
    
    - 你需要对手 bluff ≥ 100 / (100+200) = 1/3 ≈ 33.3%。
        
- pot = 100，villain bet 150（1.5x pot）：
    
    - 你需要对手 bluff ≥ 150 / (100+300) = 37.5%。
        

关键信息有两个：

1. **下注越大，你 call 所需 bluff 比例越高**；
    
2. **大底池河牌抓诈，本质是“高波动、高门槛”的决策**，不是“随手一按”。
    

### 14.3.2 《河牌抓诈进阶指南》：非稳定胜率与样本数

原文在推完“抓诈三原则”后，专门写了一段概率论：

- 设随机变量 Zn 为掷骰子 n 次的点数和；
    
- 由于期望值、方差可加性，
    
    - 当 n 足够大时，Zn 近似服从正态分布（中心极限定理）。
        

作者想强调的是：

> **只要你每一次抓诈的决策是 +EV 的，  
> 在足够多手牌后，你的总结果会收敛到“期望值附近”，  
> 中间过程再难受（连着被秀 nuts），也只是方差。**

他自己在 2019 年那手 100NL 的 T9 同花抓诈，

- 理论上 T9s 比 QQ+ 更适合抓诈，
    
- 但基于对这个对手“河牌大码偏价值”的 read，
    
    - 实战选择了弃牌，
        
    - 也明确说“这是 exploit 对手，而不是无视 GTO”。
        

你要吸收的是两点：

1. 抓诈天生属于 **“非稳定胜率”决策**：
    
    - 短期一直被秀 nuts，很正常；
        
    - 不代表你的决策是错的。
        
2. 在真实玩家池里：
    
    - 必须把 “GTO 上最好的抓诈牌”
        
    - 和 “面对当前对手、当前玩家池倾向的实际最佳决策” 区分开。

## 14.4 “抓诈三原则”：从牌例抽象出来的系统规则

《河牌抓诈进阶指南》核心就这三条（我们已经在前面章节反复用过，但这里要专门为 bluffcatch 系统化）：

> **河牌想抓诈时，你的手牌最好同时满足：**  
> 1）阻挡对手的价值范围；  
> 2）不过度阻挡对手的诈唬范围；  
> 3）能打赢对手价值范围中最弱的一部分牌。

### 14.4.1 原则一：阻挡价值范围

典型例子：文章牌例 1 中的 T9s vs QQ+/KK。

- 在对手多街 aggression 的牌面下：
    
    - T9s 可以：
        
        - 阻挡部分顺子 / set / 两对等 value 组合；
            
    - QQ+/KK 虽然绝对牌力高，
        
        - 却几乎不阻挡对手任何 value，
            
        - 面对大码下注时，经常只是付钱机器。
            

**翻译成人话：**

- 抓诈不是“拿自己最强牌去扛”，
    
- 而是“拿那些能有效减少对手 value 组合数量的牌去扛”。
    

你优先选：

- 阻挡对手 top set / top two / nut straight / nut flush 的牌；
    
- 而不是盲目拿超对 / 两对当“英雄牌”。
    

### 14.4.2 原则二：不阻挡诈唬范围

另一组例子：As7s 抓 vs 各种黑桃 bluff；Q9o 抓 vs 听顺 / 听花 bluff。

- 当你拿 As7s 时，
    
    - 会把对手大部分可能拿来 bluff 的黑桃听牌“堵死”；
        
    - 结果是：
        
        - 对手真实 bluff 组合变少，
            
        - 他在河牌下注时整体更 value-heavy，
            
        - 你 call 的 EV 自然下降。
            

**抓诈要避免的是：**

> 用那些大幅削减对手 bluff 组合的牌去 call——  
> 这样你等于在告诉自己：  
> “我知道对手 bluff 不多，但我还是想 hero 一下。”

### 14.4.3 原则三：打赢价值底端的一部分

这条容易被忽略：

- 如果对手的价值范围 **全都碾压你**，
    
    - 即便你阻挡部分 value、没阻挡 bluff，
        
    - 你被 call 到时 EV 仍然可能是负的。
        

例如文中的场景：

- As7s 面对某个 turn/river 线：
    
    - 阻挡了大量 bluff，没阻挡正确的 value，
        
    - 且被对手所有价值范围碾压，
        
    - 最终通过 solver 发现：
        
        - As7s 的 river call EV < 0。
            

**因此：**

- 好的 bluffcatch 牌：
    
    - 在对手价值范围中，
        
    - 至少能打赢价值底端的一部分（比如对手用过 thin 的 top pair 做价值）。
        

**三条一起用，才能锁定“真正该抓的那批牌”。**

## 14.5 组合数视角：把“他 bluff 多不多”拆成可估的东西

你不可能每手都做精确 combo count，但要有一套 **粗算思路**。

### 14.5.1 粗算价值组合

步骤：

1. 根据线路和牌面，写出对手可能 value 的牌型：
    
    - 比如：AA、KK、set、两对、nut flush、第二大同花……
        
2. 考虑 preflop/前几街 action，删掉不现实的：
    
    - 有些牌 preflop 会 3bet / 4bet；
        
    - 有些牌 flop/turn 一定 XR；
        
3. 对剩下的每类 value，快速估计组合数：
    
    - 同花 4 组合、set 3 组合、某些顺子 8 组合之类。
        

你不需要精确到 +/−1 组合，目的是：

> 得到“价值端总体大概是 10 几手还是 30 几手”的秤砣。

### 14.5.2 粗算 bluff 组合

通常 bluff 来自三大类：

1. 完全 miss 的同花 / 顺子听牌；
    
2. turn 之前半诈唬的 combo draw 在 river 砖掉；
    
3. 没有摊牌价值的 overcard + blocker。
    

粗算方法：

- 看牌面上有多少自然 draw（两张同花 / 三顺 / gutshot+overcard）；
    
- 看对手 flop/turn 是否激进地用这些 draw 出手；
    
- 再看 river 这张牌是否真的 **“全砖”**：
    
    - 有的 river 会让某些 draw 变成 top pair / bluffcatch，不再适合作 bluff。
        

**组合数粗算的目标：**

- 你能回答这样的问题：
    

> “在这条线里，他现实可能有的 bluff，  
> 大概是个位数、十几手，还是几乎为零？”

一旦你发现：

- 你能列出的 bluff 组合少得可怜，
    
- 而价值组合一大堆，
    
- 再看下注是 1x pot / 1.5x pot，
    
- 你就该冷静点：**这手传奇 call，很可能只是送钱。**

## 14.6 现实玩家池偏差：河牌大注普遍诈唬不足

《河牌抓诈进阶指南》的总结非常直接：

> 在 25NLz–100NLz 的弱 Reg 群体中，  
> **河牌大码下注（尤其是满池、超池）整体诈唬不足。**

作者自己：

- 明知在某些 spot 里 T9s 是优于 QQ+ 的抓诈牌，
    
- 但基于对对手“value-heavy”的 read，
    
- 选择 exploit：**在玩家池 overall underbluff 的前提下，直接弃掉。**
    

《千算抓诈锦囊·防守篇》引申出几个实战经验（抽象表达）：

1. **大多数业余玩家河牌大码下注时 bluff 远不足 GTO 要求。**
    
2. **少数极激进 / 自认为 solver 玩家会过度 bluff 某些结构。**
    
3. **线下、低额线上，都更接近“诈唬不足”而不是“诈唬超标”。**
    

这直接导出一个对你非常关键的结论：

> **默认情况下，你在河牌大码下注面对的是“低 bluff 环境”，  
> 除非你有非常强的 read 认为此人会疯狂 bluff。**

所以：

- 理论上该抓的牌（T9s 类）在现实池子里 **不一定要抓**；
    
- 你可以：
    
    - 学会“GTO 上应该抓哪类牌”；
        
    - 在读懂对手之后，有意识地少抓甚至不抓，
        
    - 这是“利用玩家池偏差”的 exploit，而不是“弃学 GTO”。

## 14.7 河牌 bluffcatch 决策 Checklist

给你一个你之后可以机械执行的 7 步流程：

1. **赔率：我需要多少胜率？**
    
    - 对手下注 B into P 底池时，你 call 需要的胜率：  
        [  
        \text{需要胜率} = \frac{B}{P + 2B}  
        ]
        
    - 记几个常用值：
        
        - 1/2 pot：≈ 25%
            
        - 2/3 pot：≈ 28–29%
            
        - pot：≈ 33%
            
        - 1.5x pot：≈ 37.5%
            
2. **玩家池/对手：他大码 bluff 多吗？**
    
    - 如果是典型“河牌大注偏价值型”：
        
        - 直接把理论需要的 bluff 比例 **打折** 看待；
            
        - 本来要 33%，现实他可能只有 10–15%。
            
    - 如果是你实锤的“over-bluff reg”：
        
        - 反而可以接受更松的抓诈 line。
            
3. **结构：这张 river 和这条线，理论上适合 bluff 吗？**
    
    - 你是否有 nuts 优势？
        
    - 对手在这条线里是否被 capped？
        
    - river 是否 blank / 利你，而非极利对手？
        
    - 如果三问都偏“否”，那对手 GTO bluff 频率就不会高。
        
4. **组合数：价值 vs bluff 大致对比**
    
    - 列出对手合理 value（AA/KK/set/两对/flush/straight）组合数，得个数量级；
        
    - 再列出合理论证下的 bluff 组合：miss draw / busted FD / SD；
        
    - 粗看一下：
        
        - bluff 是否可能接近或超过 “所需 bluff 数量阈值”？
            
5. **我的牌是否符合“抓诈三原则”？**
    
    - 阻挡了对手部分 value？
        
    - 没有严重阻挡对手 bluff？
        
    - 能打赢他价值范围底端的一部分？
        
    - 如果三条有两条是“否”，这手往往不该被拿来 hero。
        
6. **在我的整条防守范围中，这手牌处在什么层级？**
    
    - 你需要确保：
        
        - 比它更差的牌都弃了；
            
        - 它真的是你“range 顶端或中上层”的一部分。
            
    - 如果你已经用比它更好的牌弃了一堆，
        
        - 再拿它 hero，多半是情绪动作。
            
7. **接受波动：就算这是 +EV 的 call，也会经常输**
    
    - 如果你不接受“长期赢、短期连被秀 nuts”的现实，
        
    - 那就干脆把这一类边缘抓诈从你的 arsenal 中去掉，
        
    - 不要一边怕波动，一边又想做英雄。
        

Checklist 跑完，如果你仍然觉得这一枪 call 合理，

- 那就冷静按下 call，
    
- 之后好好在数据库里复盘这手，
    
- 看长期是否真如你预期那样 +EV。

## 14.8 高频场景模板：抓诈该松、该紧的典型位置

### 模板 1：IP 三枪 vs OOP 被动 call，river 砖牌大码下注

- 结构：
    
    - IP preflop aggressor，flop/turn 连续 cbet，
        
    - OOP 一路 check-call，
        
    - river 出 blank，IP pot / 1.5x pot。
        

**GTO 视角：**

- 这类结构下：
    
    - IP 保留 nuts 优势；
        
    - OOP 被强烈 capped（很多 nuts turn/river 早该 XR）；
        
    - 理论上 IP 必须带有一定比例 bluff，
        
        - 否则他的大注过度 value-heavy，
            
        - OOP 可以 exploit 弃太多。
            

**玩家池现实：**

- 多数弱 Reg 在这里 bluff 不足，
    
- 尤其线下 / micro，
    
    - 三枪大码通常“真有东西”。
        

**防守建议：**

- 对未知 / 弱 Reg：
    
    - 抓诈频率明显低于 GTO，
        
    - 只用最顶层、极优 blocker 的牌抓。
        
- 对你知道会 over-barrel 的强 reg：
    
    - 恰恰需要按 GTO 或略松抓，
        
    - 让他知道“OOP 并不随便让你三枪白打”。
        

### 模板 2：miss FD / SD 明显存在的牌面，river 对手小码下注

- 结构：
    
    - 公共牌有明显 FD/SD；
        
    - river miss；
        
    - 对手用 1/3 pot 或 1/2 pot 小码 bet。
        

**特点：**

- 小码天生价值成分偏多（“薄价值 + block bet”）；
    
- bluff 倾向性比大码低；
    
- 但同样也有一些“cheap bluff”。
    

**防守建议：**

- 面对小码，**不要把抓诈当主线**：
    
    - 你更多是在决定：
        
        - 自己的 medium strength hand 是 value 还是 bluffcatch；
            
    - 真正极端 bluffcatch（高牌）在这里不会太多。
        

### 模板 3：paired board + missed draws，IP overbet

- 结构：
    
    - board 配对 + 同花/顺子 draw 全部 miss；
        
    - IP 在 river overbet。
        

**GTO 中常见场景：**

- 进攻方 nuts 优势强（有大量 full house / trips）；
    
- 防守方 range 被 cap；
    
- 因此 overbet 中确实需要带上不少 busted draw bluff。
    

**现实中：**

- 很多玩家在这种牌面“过度尊重full house”：
    
    - 他们几乎不用 busted draw bluff，
        
    - overbet 线严重偏 value。
        

**防守建议：**

- 对这类人：
    
    - paired board + overbet = 极度谨慎的 call，
        
    - 尤其避免用“把大量 busted draw 挡没的牌”去 hero。

## 14.9 训练与复盘：让 bluffcatch 变成系统技能

### 14.9.1 Session 内刻意练习

给自己设两个任务：

1. **每个 session 至少 tag 3 手“河牌很纠结的 call/fold”。**
    
    - 记录：底池、size、对手类型、自己当时的想法。
        
2. **当场粗算一次：**
    
    - 需要胜率是多少？
        
    - 我能列出多少 bluff / value 组合？
        
    - 这手牌是否符合“抓诈三原则”？
        

哪怕当场算不清楚，

- 这个动作本身会强迫你从“感觉决策”向“结构化决策”过渡。
    

### 14.9.2 事后复盘模板

复盘时对每手 river hero call / hero fold，写：

1. 对手是谁？在你印象中是 overbluff 还是 underbluff？
    
2. 这手牌在 GTO 视角下，他大约应该有多少 bluff/value？
    
3. 在你描述的玩家池里，他现实大概有多少 bluff/value？
    
4. 你的牌满足“抓诈三原则”的多少条？
    
5. 结合赔率（所需 bluff 频率），
    
    - 这手 call/fold 是倾向 +EV 还是—EV？
        
6. 若再次遇到类似结构 & 对手，
    
    - 你是否要系统性调整：更常抓/更少抓？
        

### 14.9.3 长期训练方向

- 建立一个 “bluffcatch 标签”：
    
    - 所有河牌 hero call / hero fold 的手牌都打标签；
        
    - 定期检视：
        
        - 你的 hero call 赢率是否远低于“理论所需胜率”？
            
        - 在 underbluff 池子里，你是否还在硬扛？
            
- 用 solver（或标准路线）对一些典型结构做 drill：
    
    - 比如：IP 三枪 vs OOP call；3BP pot；paired board；四同花面；
        
    - 让自己熟悉：
        
        - 在这些标准牌面上，
            
        - 哪些手牌在 GTO 里是 pure call / mix call / pure fold。

## 14.10 常见错误（至少 7 条）

1. **用“绝对牌力”而不是“相对范围位置”抓诈**
    
    - 觉得 QQ/KK 很大，就拿来 call 大码，
        
    - 完全不看自己在整个范围中是否已经“被对手 value 压着打”。
        
2. **忽视玩家池 underbluff 现实，照抄 solver MDF**
    
    - 为了看起来像 reg，
        
    - 在明显诈唬不足的环境中硬守最低防守频率，
        
    - 结果长期拿 marginal hand 去接大注，白送 EV。
        
3. **不会用“抓诈三原则”筛牌**
    
    - 拿一堆阻挡对手 bluff 的牌去 hero，
        
    - 或者拿完全不阻挡 value 的牌去扛，
        
    - 组合意义上完全站在自己对立面。
        
4. **不算赔率，不知道自己需要多少 bluff 频率**
    
    - 面对 1.5x pot 还在说“我只要赢一次就回本”，
        
    - 完全无视自己实际上需要对手至少 37.5% bluff 才不亏。
        
5. **线条不连贯：前面被动、河牌突然变成英雄**
    
    - flop/turn 一路被动 flat，
        
    - river 面对极强线还要 hero，
        
    - 在范围里自己已被 cap，却假装自己在顶端。
        
6. **把单次结果当成策略评判标准**
    
    - 一次正确的 +EV 抓诈碰上对手 value 就否定整套逻辑，
        
    - 忽略了河牌抓诈属于样本需要非常多的“非稳定胜率”决策。
        
7. **只记得自己“抓错时的疼”，忘记没抓到的巨大 EV**
    
    - 只记得某次 hero call 输了大锅，
        
    - 完全不复盘那些对手明显 overbluff、你却怂 fold 的 spot，
        
    - 导致策略越来越保守，长期只剩“惧怕波动”，而不是“追求 EV”。
